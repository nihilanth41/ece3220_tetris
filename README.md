# Tetris

# Installation

```
git clone https://github.com/nihilanth41/ece3220_tetris.git
cd ece3220_tetris/src 
make
./tetris.bin
```

# How to play

'j' - Move left   
'k' - Move right  
'l' - Rotate   
'q' - Quit
